\subsection{Reuniones}

\subsubsection{Buenos días}

Todos los días a las 9:30 de la mañana se reúnen un miembro de cada departamento para intercambiar información acerca del funcionamiento de la fábrica. Se trata de una reunión muy breve que se desarrolla en 5 minutos y de pie para que sea lo más breve posible. A continuación, detallo la contribución de cada departamento:

\begin{itemize}[noitemsep,topsep=0pt,parsep=0pt,partopsep=0pt]
	\item Producción
	\begin{itemize}
		\item Cantidad de piezas fabricadas el día anterior.
		\item Máquinas paradas.
	\end{itemize}
	\item Compras
	\begin{itemize}
		\item Nuevos proveedores.
		\item Detalles importantes acerca de las nuevas piezas.
	\end{itemize}
	\item Ingeniería de Producto Avanzada
	\begin{itemize}
		\item Llegada de nuevas ordenes de fabricación.
	\end{itemize}
	\item Ingeniería de Fabricación Avanzada
	\begin{itemize}
		\item Desarrollo de nuevas piezas.
	\end{itemize}
	\item Calidad
	\begin{itemize}
		\item Fallos en la producción de piezas.
		\item Back orders.
	\end{itemize}
	\item Informática
	\begin{itemize}
		\item Nuevas funcionalidades añadidas.
		\item Cambios en equipos informáticos que afectan a la producción.
	\end{itemize}
	\item Seguridad y Medio Ambiente
	\begin{itemize}
		\item Problemas de seguridad encontrados.
		\item Incumplimientos de la normativa de seguridad laboral.
		\item Cursos de seguridad.
		\item Accidentes.
	\end{itemize}
	\item Recursos Humanos
	\begin{itemize}
		\item Ausencias de personal.
		\item Personal de empresas subcontratatadas.
		\item Nuevo personal.
		\item Visitas a fábrica.
	\end{itemize}
\end{itemize}

\subsubsection{Control de progreso}

He desarrollado 2 reuniones con el resto de miembros de IT para mostrar mi progreso con los proyectos. La primera de ellas fue para ver los diferentes planteamientos que había estudiado para la medición usando visión artificial. La segunda, fue para explicar todos los proyectes que había desarrollado durante mi estancia, para su posterior mantenimiento y desarrollo.
